#1. Agricultura de Precisão e Tecnologia (O Agro Digital)
Este tema foca em como a tecnologia moderna ajuda o agricultor a produzir mais gastando menos recursos.

Ideia do Jogo: O jogador gerencia uma fazenda tecnológica e precisa decidir onde investir o seu dinheiro.

Decisões Detalhadas para o Código:

Escolha Sustentável: Investir em sensores de umidade no solo (gasta moedas, mas economiza água e aumenta a sustentabilidade).

Escolha Ruim: Irrigar a plantação inteira no modo manual, gastando água em excesso para economizar moedas a curto prazo.

Escolha Sustentável: Usar imagens de satélite/drones para aplicar fertilizante apenas onde a planta precisa.

Visual: O fundo verde pode ter pequenos ícones de antenas, tratores autônomos ou drones sobrevoando.
